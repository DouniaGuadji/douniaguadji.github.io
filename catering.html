Airline Catering Weight Optimizer

Accueil Projet Design

Contexte

Optimiser la masse de base d'un avion via la gestion des prestations catering, pour réduire la consommation de carburant.

Approche

  • UI React simple (sélection type avion / appareil / prestation).
  • API factice pour calculer la masse totale.
  • Tests end-to-end basiques.

Résultats & Démo

Règles métier simplifiées, calculs reproductibles, interface claire.
